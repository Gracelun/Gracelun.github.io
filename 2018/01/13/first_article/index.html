使用Hexo在Github架BLOG

發表於 2018-01-13

1.Hexo是什麼?

Hexo 是一個快速、簡單且強大的網誌框架。
Hexo 使用Markdown（或其他渲染引擎）解析您的文章，並在幾秒鐘內，透過漂亮的主題產生靜態檔案。

2. 安裝需求

安裝Hexo前，檢查電腦是否已有下列軟體

  • Node.js
  • Git

3. Hexo安裝

3.1 建立

_config.yml

網頁配置檔案，可以在此配置大部分的設定

4. 部署至Github

4.1創建Github Pages

註冊Github帳號，創建跟帳號同名的repo

username.github.io

創建完後，在此repo頁面點擊：

Settings>Github Pages>Launch automatic page generator

這樣就生成Github Pages了。

4.2 設定Hexo

在Hexo文件夾中找到_config.yml檔案，找到deploy:，修改成以下：

deploy:
type: git
repo: git@github.com:/.github.io.git
branch: master
